    10‚Üí## Architecture Decisions
    11‚Üí- **Single route constraint:** All UI lives in `/` (src/app/page.tsx) with state-based view switching via a Zustand store. API routes under `/api/*` handle all backend logic.
    12‚Üí- **Auth:** NextAuth.js v4 with Credentials provider, bcrypt password hashing, role-based session.
    13‚Üí- **RBAC roles:** SUPER_ADMIN, ADMIN, MANAGER, EXECUTIVE
    14‚Üí- **Audit trail:** Every mutation logged to AuditLog table (immutable).
    15‚Üí- **Theme:** Industrial Blue (#0EA5E9 / sky-600) + Charcoal (#1F2937 / gray-800) + White, with dark mode via next-themes.
    16‚Üí- **AI:** z-ai-web-dev-sdk LLM for lead scoring, recommendations, forecasting narrative.

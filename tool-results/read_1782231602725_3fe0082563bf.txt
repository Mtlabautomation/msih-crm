    10тЖТ    10тЖТ## Architecture Decisions
    11тЖТ    11тЖТ- **Single route constraint:** All UI lives in `/` (src/app/page.tsx) with state-based view switching via a Zustand store. API routes under `/api/*` handle all backend logic.
    12тЖТ    12тЖТ- **Auth:** NextAuth.js v4 with Credentials provider, bcrypt password hashing, role-based session.
    13тЖТ    13тЖТ- **RBAC roles:** SUPER_ADMIN, ADMIN, MANAGER, EXECUTIVE
    14тЖТ    14тЖТ- **Audit trail:** Every mutation logged to AuditLog table (immutable).
    15тЖТ    15тЖТ- **Theme:** Industrial Blue (#0EA5E9 / sky-600) + Charcoal (#1F2937 / gray-800) + White, with dark mode via next-themes.
    16тЖТ    16тЖТ- **AI:** z-ai-web-dev-sdk LLM for lead scoring, recommendations, forecasting narrative.
